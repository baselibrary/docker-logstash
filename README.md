## ThoughtWorks Docker Image: logstash

[![](http://dockeri.co/image/baselibrary/logstash)](https://registry.hub.docker.com/u/baselibrary/logstash/)

### Base Docker Image

* `latest`: logstash 2.0
* `2.0`   : logstash 2.0
* `1.5`   : logstash 1.5
* `1.4`   : logstash 1.4

### Installation

    docker pull baselibrary/logstash

### Usage

    docker run -it --rm baselibrary/logstash
